XState: State Machines for JavaScript

I've been exploring a library called XState that brings state machines to JavaScript. One of the benefits of using a state machine is that it makes frontend code easier to test. It also makes the creation of state charts easy, so you can understand how your components interact without reading code or documentation!

Reflection: 15 years of JavaScript

I've been using JavaScript for almost 15 years now. I started my career as a web developer in 2007, on the eve of iPhones and modern web frameworks. Back then, Internet Explorer still had a dominant share of the browser market and the best library we had for building rich user interfaces was jQuery UI. There was an alternative option in Macromedia Flash, but it was eventually killed of when Apple made the decision to drop Flash support from iOS Safari.
