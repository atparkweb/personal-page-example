Bio

I was born and raised in Saint Paul, Minnesota, USA. My background is in art and design, but I have always been insterested in computers as a hobby. I started my professional career as a web designer in 2007, and moved into web development through programming ActionScript, JavaScript, and C# for web applications. Since then I have experienced working with a broad range of web technologies and programming languages.

I currently work at Code Chrysalis, a code bootcamp based in Tokyo, Japan, as the lead instructor of an immersive 3 month program. I enjoy teaching because I like to talk about all things code related, and I like to help people discover how fun and interesting programming can be.
